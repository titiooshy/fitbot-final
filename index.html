    WE MADE A MACHINE THAT
    PERFORMS CHORES. YES, CHORES.

    The FitBot Vending Machine is a custom-made robot that makes simple chores simple. Set your preferred task to be completed right before your eyes by using our app to place an order.

    Our goal for this project was to build an aesthetically pleasing method of automating household chores, allowing for customer customization. The FitBot features advanced sensors, motion controls, and a custom chassis. Our website gives many customization options for tasks and allows for a lot of user interactivity.
